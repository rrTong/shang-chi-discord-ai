## Keyword Triggers

Shang-Chi bot will only respond when the message contains any of the following:

- `@Shang-Chi`
- Xu, 徐
- Shang, 尚, シャン, 샹, Шан, شانج
- Chi, 氣, 气, 気, チー, 치, Чи, شي
- Shaun, Shawn, Sean
- legend, 傳奇, 传奇, 伝説
- ring, 環, 环, リング
- wenwu, 文, 武
- katy
- xialing, 夏, 靈, 灵
- dragon, 龍, 龙, 竜
- master, 師父, 师父, 師匠
- kung fu, 功夫
- ta lo, 大羅, 大罗
- 🐉, 🐲, 💍
